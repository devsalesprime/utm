Bem-vindo, Agente do projeto UTM Generator. 

Ao realizar tarefas neste repositório, por favor, sempre inicie lendo (se já não estiver explícito na sua cache):
- `.context/index.toml` para obter instruções de roteamento dos próximos arquivos baseados no intuito do usuário.
- `.context/project.toml` para lembrar as diretivas do servidor e do aplicativo.
- `.context/rules.md` para práticas de código.

Este projeto aplica o modelo aberto [DotContext](https://github.com/gmasson/dotcontext) para portabilidade e memória limpa entre os assistentes de IA.

Boa sorte!
